### 具体玩法
* 每个课题一周，每周由@麦梓出部分demo来抛砖引玉
* 组员进行研究学习
* 同时设计湿们@幽琪把想要实现的想法设计出来让组员去实现
* 组员去实现设计湿们的想法和自己的想法，并把代码贡献到代码库和demo平台
* 设计湿们设计个demo展现平台来展现大家智慧的结晶
* 大家一起来出总结和分享，总结可以包括：技术要点、效果展示、在业务中的应用、兼容性、参考资料
